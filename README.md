# TravelSNS（開発中）

このアプリは「旅のルート・写真・思い出を共有する SNS」です。  
現在は **ログイン・新規登録・投稿機能のみ動作**しています。  
その他の機能は順次開発予定です。

---

##  開発状況（Progress）

###  完成済み
- ユーザー登録
- ログイン
- 投稿作成（画像アップロード対応）

###  開発中
- プロフィール編集
- 投稿検索
- タグ機能
- おすすめ投稿
- コメント機能
- 通知機能

---

## 🧪 現在動作する API

- `POST /api/auth/register` – 新規登録  
- `POST /api/auth/login` – ログイン  
- `POST /api/posts` – 投稿作成
